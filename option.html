CSSViewly Extension

CSSViewly helps you quickly inspect CSS properties of elements on any web page.

Activate the extension, hover over elements, and instantly see key style information without opening developer tools.

How to Use

  • Click the CSSViewly icon in your browser toolbar to activate or deactivate the viewer.
  • When active, simply hover your mouse over any element on the page.
  • A small overlay panel will appear displaying basic CSS information (like colors, fonts, dimensions, etc.) for the hovered element.

Privacy

Your privacy is important. CSSViewly needs permission to access web page content solely to inspect element styles.

The extension does not collect, store, transmit, or share any personal data, browsing history, or user activity. All processing happens locally within your browser.
